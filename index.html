NiiVue Demos for EEG, iEEG and source space connectivity

NiiVue is a web-based visualisation tool for neuroimaging that can run on any operating system and any web device (phone, tablet, computer).

This page is to test whether one can use this tool for visualising EEG, iEEG and source-based connectivity interactively on any web browser.
